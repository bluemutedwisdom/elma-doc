First Boot¶

Before first boot it is strongly recommend that the Hardware (CMOS/BIOS) clock is set to UTC/GMT!

  • Clock and time adjusting

On the first boot, you will be asked to configure several items:

  • Hard Disk Preparation (Preload ISO image only)
  • Language
  • Keyboard Layout
  • Hostname and Domain Name
  • Network Configuration
  • Clock and Time Zone
  • Set the ROOT password

Installation settings¶

  • All images boots into a simple graphical environment.

  • The Preload ISO image prompts for confirmation before writing the image onto your hard disk. Select “Yes” to continue with automatically format and partition your hard disk.

  • After several screens of partitioning we are ready for the initial setup.

  • Select your preferred language

  • Select your keyboard layout

  • Enter your hostname and domain name

  • Select “Change” to edit networking

  • Select “Network Interfaces...”

  • Select “Edit”

  • Select “Statically assigned IP Address”. Enter your static IP address, subnet mask, and hostname. Then select “Next”.

  • Select “Hostname/DNS”. Enter your DNS servers, and select “OK”

  • Select “Routing”. Enter your default gateway, and select “OK”

  • Confirm your settings and select “Next”

  • Wait for the next prompt

  • It is strongly recommend that the Hardware (CMOS/BIOS) clock, system and local time is set to UTC/GMT!

  • Select your time-zone and select “Next”

  • Your appliance is now ready for ELMA applications to be installed!

  • For good measure login and reboot your appliance.
